# react-native-quick-pinyin

A module parses chinese to pinyin for React Native(jsi)

## Installation

```sh
npm install react-native-quick-pinyin
```

## Usage

```js
import Pinyin from "react-native-quick-pinyin";

// ...

const result = await Pinyin.multiply(3, 7);
```
